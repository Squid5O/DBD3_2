# DBD(댕방댕)
-------------------
## 기간
2024년 1월 29일 ~ 2월 19일

-------------------
## 개요
악마의 생존자 사냥 게임(DeadByDayLight 카피캣 프로젝트)

-------------------
## 개발
언리얼 블루프린트(5.3v)

-------------------
## 기능
- 생존자 캐릭터 Behavior Tree 이용한 AI 구현
- 생존자 상태 변화시 UI 변화 연동
- 플레이어(살인마) 일반 공격 , 특수 공격 구현
- 생존자 발바닥 Decal 이미지로 추적 기능
- 엔진 기동을 통한 맵 이벤트 구현
- 특수 , 일반 공격 Hit 발생시 생존자 상태 변화
- 플레이어 방해하는 맵 구조물 생성 추가
